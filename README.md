# JustSimpleChat
**JustSimpleChat** is an **Innovative**, **Easy**, **Safe** and **Fast** Chat system. 
JustSimpleChat does not use any **Google Cookie** and is developed on **PHP** and **JS** and has a very **Simple and Light** design.
